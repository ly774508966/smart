### 项目结构
``` lua
    web-api
    ├── assembly -- 打包脚本
    ├── java -- 代码
    ├    ├── config -- 配置
    ├    ├── controller -- 控制层
    ├    ├── core -- 框架代码
    ├    ├── dto -- 传输对象
    ├    ├── filter -- 拦截器
    ├    ├── model -- 领域模型
    ├    ├── Application.java -- 启动入口
    ├── lib -- 本地jar包
    ├── resource -- 资源
    ├    ├── public -- 静态资源
    ├    ├── template -- 模板资源
    ├    ├── application.properties -- 配置文件
    ├    ├── logback.xml -- 日志配置
```
